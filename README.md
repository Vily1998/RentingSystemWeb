# RentingSystemWeb
一、运行环境配置说明

	•开发语言：前端：HTML、JSP、CSS；后端：Java
  
	•IDE: Eclipse IDE for Java EE Developers
  
	•OS: Windows 10
  
	•数据库：mysql
  
	•开源框架：
     前端：Vue；
     后端：Spring、SpringMVC、Mybatis
  
	•其它关键技术：前端：element-ui后端：JdbcTemplate
  
	•工程名称：前端：租房系统 后端：renting_system
  
	•访问URL：http://localhost:8002/

      密码：admin
